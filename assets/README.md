# Assets Directory

Static assets for Swiftbird Internet Pvt Ltd documentation site.

## Structure

- `css/` - Stylesheets
- `js/` - JavaScript files  
- `images/` - Photos, logos, graphics
- `icons/` - Icons and favicons (generated from logo.png)
- `fonts/` - Web fonts
- `documents/` - PDFs and downloads

## Favicon Files

Generated from `images/logo.png`:

- `favicon.ico` - Traditional multi-size favicon
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon  
- `apple-touch-icon.png` - 180x180 Apple touch icon
- `icon-192x192.png` - PWA icon (192x192)
- `icon-512x512.png` - PWA icon (512x512)

## Usage

Reference assets with relative URLs:

- Images: `/assets/images/filename.jpg`
- Icons: `/assets/icons/filename.svg`
- Documents: `/assets/documents/filename.pdf`

## Guidelines

- Optimize images for web performance
- Include alt text for accessibility
- Follow naming conventions: lowercase, hyphens
- Review process required for new assets
