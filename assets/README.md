# Assets Directory

Static assets for Swiftbird documentation site.

## Structure

- `css/` - Stylesheets
- `js/` - JavaScript files  
- `images/` - Photos, logos, graphics
- `icons/` - Icons and favicons
- `fonts/` - Web fonts
- `documents/` - PDFs and downloads

## Usage

Reference assets with relative URLs:

- Images: `/assets/images/filename.jpg`
- Icons: `/assets/icons/filename.svg`
- Documents: `/assets/documents/filename.pdf`

## Guidelines

- Optimize images for web performance
- Include alt text for accessibility
- Follow naming conventions: lowercase, hyphens
- Review process required for new assets
